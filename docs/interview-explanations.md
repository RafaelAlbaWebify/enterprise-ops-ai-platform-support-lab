# Interview Explanations

## Main explanation

I built this homelab to connect my IT Operations and Application Support background with AI Platform Operations and modern infrastructure support.

The focus is not advanced engineering. The focus is operational readiness: incident triage, identity and access troubleshooting, API error interpretation, pipeline log awareness, monitoring, escalation, documentation and continual service improvement.

## What I can honestly say

- I can troubleshoot support-level Microsoft 365, identity and access issues.
- I understand the difference between authentication and authorization.
- I can interpret common API errors such as 400, 401, 403, 429 and 500.
- I can read basic pipeline logs and collect evidence before escalation.
- I can create runbooks, KB articles and incident notes.
- I can use PowerShell and Python for basic support automation.
- I understand model lifecycle concepts at support level.

## What I should not claim

- I am not a senior DevOps engineer.
- I am not an Azure AI architect.
- I am not an MLOps engineer.
- I am not a VMware architect.
- I am not a network engineer.
- I am not a senior Python developer.
