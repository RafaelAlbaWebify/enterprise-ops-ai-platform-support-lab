# INC-004 — Microsoft 365 User Access Blocked

## Summary

User cannot access Microsoft 365 service.

## Impact

Single user affected. Productivity impacted.

## Symptoms

- User reports access denied.
- Service works for other users.
- Issue affects browser and desktop application.

## Initial checks

- Confirmed user account.
- Confirmed affected service.
- Requested screenshot.
- Checked whether user recently changed team or role.
- Checked license and group membership conceptually.
- Checked service health conceptually.

## Possible causes

- License issue
- Group membership issue
- Conditional Access-style restriction
- Client cache/session issue
- Tenant service issue

## Escalation

Escalate to Microsoft 365 / identity admin team if policy or license assignment must be changed.
